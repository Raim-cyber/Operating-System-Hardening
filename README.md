# Operating System Hardening

## Overview
Academic operating system hardening project performed on Kali Linux to improve system security through configuration and protection mechanisms.

## Objectives
- Improve operating system security
- Reduce attack surface
- Apply secure system configurations

## Security Tasks Performed
- System Update
- UFW Firewall Installation
- Firewall Rules Configuration
- SSH Hardening
- Fail2Ban Installation
- File Permission Management

## Technologies Used
- Kali Linux
- UFW
- SSH
- Fail2Ban
- Linux Commands

## Security Concepts
- Access Control
- System Hardening
- Secure Configuration
- Authentication Security

## Included Files
- Operating System Hardening Report
